This is a newsify app Used to Show News. 

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

## Features

* Searching And Filtering the news.
* Categoriesing The News.'
* Bookmark/Unbookmark The News.
* Get Single News.
* Use NewsData Api to Get All News.
